## Descripcion
Este Proyecto busca ser una Aplicacion Movil donde los Usuarios puedan tomar notas de forma rapida e intuitiva para tenerlos siempre al alcance asi como notas rapidas, alarmas y lista de tareas.
Este es un Pequeño proyecto que no busca realizar operaciones si no enfocarse en accesos rapidos, directos, simples y faciles de utilizar.
